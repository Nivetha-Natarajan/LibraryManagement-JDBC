# Library Book Lending System

A console-based Java JDBC application for managing library books, book issue transactions, and returns using MySQL.

## Features

* Add New Book
* View Book Details
* View All Books
* Issue Book to Students
* Return Issued Books
* Remove Books
* Input Validation and Exception Handling

## Technologies Used

* Java
* JDBC
* MySQL
* Eclipse IDE

## Project Structure

* `com.library.app` – Main Application
* `com.library.bean` – Entity Classes
* `com.library.dao` – Database Operations
* `com.library.service` – Business Logic
* `com.library.util` – Utility & Exception Classes

## Database Tables

* `BOOK_TBL`
* `ISSUE_TBL`
